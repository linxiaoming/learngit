git学习命令
git init 
git add 文件名
git commit -m ‘备注说明’
git log 
git log -pretty=oneline
git reset --hard HEAD^
git reset --hard HEAD^^
git reset --hard HEAD~100
git reflog 
git reset -hard 版本id
与其他版本控制器不一致的是git有缓存区
git status 查看状态
git diff HEAD -- test.txt 比较工作区的与版本库最新版的区别
git checkout HEAD -- test.txt 撤销修改
git rm 文件名 删除版本库相应的文件
git branch fz 创建分支
git checkout fz 切换分支
git merge fz 合并分支.
